Up and running in
three steps.

  1. 1

    Install the extension

    Add SN Launcher from the Chrome Web Store or Firefox add-ons store. Works on Edge and all Chromium browsers too.

  2. 2

    Open ServiceNow & press the shortcut

    Navigate to any ServiceNow instance and hit Ctrl + Shift + L or your customized shortcut to open the command palette.

  3. 3

    Type and go

    Search across all your menus instantly. Select a result and you're there. That's it.
